What exactly is a Pomodoro Timer?

A Pomodoro Timer is a time management method that breaks down work into 25-minute intervals. After each interval, you take a 5-minute break. After every 4 work interval, you take a longer 15 to 30 minute break.

This application allows you to:

  • Set a Pomodoro Timer.
  • Create tasks that you have to complete and the number of interval it would take to finish.
  • Search for specific tasks using their name or interval.
  • Edit and delete specific tasks that you've created
